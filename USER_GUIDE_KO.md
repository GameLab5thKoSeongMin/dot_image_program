# 사용자 가이드

## 1. 프로그램 개요
이 프로그램은 PNG, JPG, JPEG 이미지를 입력받아 32x32 픽셀 아이콘을 만드는 로컬 웹앱입니다.

단순히 이미지를 32x32로 줄이는 방식이 아니라, 원본 이미지를 32열 x 32행 타일로 나눈 뒤 각 타일의 median 색상값을 계산해서 결과 픽셀을 만듭니다.

## 2. 실행 방법
가장 간단한 방법은 `index.html` 파일을 브라우저에서 직접 여는 것입니다.

브라우저 환경에서 직접 열기가 제한되면 프로젝트 폴더에서 다음 명령을 실행할 수 있습니다.

```bash
python -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000/`을 엽니다.

## 3. 사용 방법
1. `index.html`을 엽니다.
2. 파일 선택 버튼을 누르거나 이미지를 드래그해서 입력 영역에 놓습니다.
3. 좌측 상단에서 원본 이미지 미리보기를 확인합니다.
4. 우측 상단에서 생성된 32x32 아이콘 미리보기를 확인합니다.
5. 우측 하단의 다운로드 버튼으로 PNG 결과물을 저장합니다.

## 4. 지원 파일 형식
- PNG
- JPG
- JPEG

## 5. 결과물 사양
- 크기: 32x32 px
- 형식: PNG
- 투명 PNG 배경 지원
- 파일명: 원본 파일명이 있으면 `original_name_32x32.png` 형식으로 생성

## 6. 주의사항
- 매우 큰 이미지는 브라우저 메인 스레드에서 처리되므로 잠시 멈춘 것처럼 보일 수 있습니다.
- 현재 버전의 출력 크기는 32x32로 고정되어 있습니다.
- 알고리즘 검증은 `tests/test-cases.html`에서 다시 실행할 수 있습니다.
